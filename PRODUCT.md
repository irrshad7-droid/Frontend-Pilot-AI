# Product

## Register

product

## Platform

web

## Users
Developers, QA engineers, and hackathon judges/presenters. Developers and QA engineers need quick diagnostics, error maps, and surgical code diff previews. Hackathon judges require a high-impact, 15-second visual comprehension of the autonomous engineering loop.

## Product Purpose
FrontendPilot AI is an autonomous quality engineering agent that runs target web applications in Playwright, parses runtime errors, maps failures to codebase JSX nodes via tree-sitter AST queries, and synthesizes regression-free patches.

## Positioning
"Your frontend broke. Watch AI fix it." — An autonomous, end-to-end frontend repair agent that does not just alert, but actively investigates and repairs bugs.

## Brand Personality
Surgical, precise, and highly intelligent. The interface feels premium and clean, drawing design aesthetic alignment from developer-centric tools like Vercel, Linear, Cursor, Raycast, and Warp.

## Anti-references
Flat, generic administrative templates (such as Jira, Grafana, Kibana, or standard bootstrap boxes). Overly marketing-focused SaaS websites with zero technical visibility, or plain console terminals lacking visual layouts.

## Design Principles

1. **Autonomous Agency**: Highlight the active agent loop (crawling, AST mapping, reasoning, testing) with dynamic indicators to generate suspense and highlight the AI's intelligence.
2. **Surgical Precision**: Design code changes, AST nodes, and verification logs as surgical block-insertions rather than text dumps.
3. **High Technical Credibility**: Always display real logs, trace steps, and network snapshots alongside visuals to prove E2E functionality to developers.
4. **Focused Guidance**: Structure elements hierarchically so that secondary log details never compete with the hero investigation panel.

## Accessibility & Inclusion
Ensure high color contrast (ratio ≥ 4.5:1) for developer text strings and respect systems prefers-reduced-motion queries by disabling animation sweeps when requested.
